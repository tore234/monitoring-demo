# monitoring-demo

Repositorio de práctica para **monitoreo de aplicaciones en Python**.  
La idea de este proyecto es tener una pequeña aplicación de ejemplo sobre la cual podamos hacer ejercicios de:

- Monitoreo de estado de la aplicación  
- Manejo de logs y errores  
- Pruebas básicas del comportamiento del código  

---

## Contenido del repositorio

- **`main.py`**  
  Script principal de la demo. Aquí se concentra la lógica de la aplicación que será “monitoreada” (funciones, procesos o simulaciones que puedes ir ampliando según la práctica).

- **`test_app.py`**  
  Archivo de pruebas. Contiene pruebas de ejemplo (o el espacio para crearlas) para validar que la aplicación funciona como se espera.

- **`README.md`**  
  Documento con la explicación general de la práctica, objetivos y pasos para ejecutar el código.

---

## Objetivos de la práctica

- Comprender los conceptos básicos de monitoreo de aplicaciones.  
- Practicar la organización de un proyecto pequeño en Python.  
- Ejecutar y probar un script desde la línea de comandos.  
- Preparar la base para integrar herramientas de monitoreo (logs, métricas, alertas, etc.) más adelante.

---

## Requisitos

- Python 3.x instalado  
- (Opcional) `virtualenv` o entorno virtual similar  
- (Opcional) `pytest` u otro framework de pruebas si deseas ejecutar `test_app.py` como pruebas automatizadas
